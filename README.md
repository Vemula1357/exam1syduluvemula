> ## Overview
>
> This project is a fully interactive web application built with Streamlit that allows users to explore a real automobile dataset. Users can filter by make, year, price, and fuel type, instantly generating updated charts and summary statistics  demonstrating production-ready data app development skills.
>
> ---
>
> ## Key Features
>
> - Live filtering and dynamic chart updates with no page reloads
> - - Multi-variable exploration: price ranges, fuel types, makes, mileage
>   - - Summary statistics and trend visualizations auto-generated from filters
>     - - Clean, deployable Streamlit app structure with modular code
>       - - Real-world dataset (Automobiles.csv) with 200+ records
>
